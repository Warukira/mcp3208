enum MCP3208::Channel

The ADC has 8 inputs, usable as 8 individually single inputs with reference to analog ground or 4 differential inputs. The polarity of each differential input can be configured.

bits 4 3 2 1
- type ch ch ch
- description :–:
type 1 bit for single(1) or differential(0) input
ch 3 bit for channel number selection

Calibrates read timing using the supplied channel. A calibration should be performed after evey SPI frequency changes or other events that could have an impact on the sampling speed. The SPI interface must be initialized and put in a usable state before calling this function.
